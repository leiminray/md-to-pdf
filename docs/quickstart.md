# Quickstart

## Basic Rendering

```bash
md-to-pdf input.md -o output.pdf
```

## With Brand

```bash
md-to-pdf input.md -o output.pdf --brand idimsum
```

## With Watermark

```bash
md-to-pdf input.md -o output.pdf --watermark-user alice@example.com
```

## Deterministic Output

```bash
md-to-pdf input.md -o output.pdf --deterministic
```
