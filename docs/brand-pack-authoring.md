# Brand Pack Authoring

Brand packs define the visual identity for PDF renders. See `examples/brands/idimsum/` for reference.

## Schema (v2.0)

```yaml
spec_version: "v2.0"
metadata:
  name: "Brand Name"
  description: "Description"
assets:
  logo: "logo.png"
  icon: "icon.png"
footer:
  text: "© 2024"
  font_size: 8
```
