# CJK 全角/半角字符全面验证

测试 md-to-pdf 对各种 Unicode 字符的渲染：不应出现方块（tofu）或乱码。

## 1. 全角标点符号

中文常用全角标点：
- 逗号：你好，世界
- 句号：测试。结束
- 冒号：键：值
- 分号：第一；第二
- 问号：是吗？
- 感叹号：好啊！
- 引号：「直角引号」、『弯引号』
- 括号：（圆括号）、【方括号】、《书名号》
- 顿号：苹果、香蕉、橘子

## 2. 半角标点符号

英文标点（应正常显示）：
- Comma, period, colon: semicolon; question? exclamation!
- Quotes: "double" 'single' `code`
- Parens: (round) [square] {curly}
- Email: sophie@example.com, alice.bob@idimsum.com
- URL: https://example.com/path?key=value

## 3. 中英文混排

| 中文标签 | 英文标签 | 描述 |
|---------|---------|------|
| 用户名： | username: | sophie |
| 邮箱： | email: | sophie@idimsum.com |
| 部门： | department: | Engineering 工程部 |
| 电话： | phone: | +852 5579 2115 |

## 4. 中日韩 (CJK) 文字

- 简体中文：你好世界，分支运营 AI 机器人产品说明书
- 繁體中文：你好世界，分支運營 AI 機器人產品說明書
- 日本語：こんにちは世界。これは日本語のテストです。
- 한국어：안녕하세요 세계. 이것은 한국어 테스트입니다.

## 5. 数字与符号

- 全角数字：０１２３４５６７８９
- 半角数字：0123456789
- 全角字母：ＡＢＣａｂｃ
- 半角字母：ABCabc
- 货币：¥100、$200、€300、£400、₩500

## 6. 特殊符号

- 数学：± × ÷ ≈ ≠ ≤ ≥ ∞ √ ∑ ∫
- 箭头：← → ↑ ↓ ⇒ ⇐ ⇔
- 几何：○ ● △ ▲ □ ■ ★ ☆
- 度量：°C 25° 90% ‰

## 7. Emoji（限制：可能显示为方块，需要 Emoji 字体）

- 表情：😀 😂 🥰 🤔
- 物品：🚀 💻 📱 🖥️
- 标志：✅ ❌ ⚠️ ✨
- 旗帜：🇨🇳 🇭🇰 🇯🇵 🇺🇸

## 8. 代码块（保持等宽）

```python
def 你好(name: str) -> str:
    """问候函数：返回带中文的问候语。"""
    return f"你好，{name}！"

# 测试：
print(你好("世界"))  # 输出：你好，世界！
```

## 9. 引用块

> 引用中包含全角字符：这是一段说明文字。
> 第二行：测试换行后的渲染。
> 第三行：包括标点（句号。）和符号（百分号％）。
